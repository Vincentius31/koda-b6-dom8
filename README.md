# Program Konversi Suhu dengan Metode Form Serialization

Aplikasi ini adalah aplikasi untuk mengkonversi suhu dari celcius ke fahrenheit, reamur, dan kelvin. Aplikasi ini bisa menyimpan data secara real time jika ingin melakukan input secara berkali-kali sampai menutup aplikasi ini.

## Tampilan Program

![Tampilan Awal Program](readme-img/127.0.0.1_3000_index.html%20(1).png)

## Cara Penggunaan Aplikasi
1. Lakukan clone pada repo ini
2. Buka file index.html
3. Masukan angka ke dalam form
4. Pencet Konversi 
5. Lihat hasil langsung pada table yang disediakan
